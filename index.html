Automation Framework for Login Functionality

Designed and implemented a robust Selenium Automation Framework using Java, TestNG, and Maven to validate application login functionalities. By adopting the Page Object Model (POM) architecture, the test logic was effectively isolated from UI elements, ensuring high-quality software delivery. The framework utilizes structured locators, explicit waits for dynamic synchronization, and custom assertions to ensure flawless test execution and accurate reporting.

Java / Selenium / Maven Code

RFID Based Women Safety System

An RFID-Based Women Safety System is an IoT-enabled smart security wearable designed to provide immediate protection and location tracking for women in distress or emergency situations. Instead of relying on a smartphone—which might be difficult to access during danger—this system utilizes a discreet, compact RFID tag (embedded in a keychain, ring, or watch) and a main wearable module containing an RFID reader, microcontroller, GPS, and GSM modules.
